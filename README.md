# 🖼️ ФотоКонвертер Pro

**Офлайн-конвертер изображений с современным 3D-интерфейсом**

![HTML5](https://img.shields.io/badge/HTML5-100%25-orange)
![JavaScript](https://img.shields.io/badge/JavaScript-Pure-yellow)
![Offline](https://img.shields.io/badge/Offline-Ready-green)
![License](https://img.shields.io/badge/License-Free-blue)

## ✨ Возможности

### 🖼️ Конвертация
- Поддержка форматов: **PNG, JPEG, WEBP**
- Регулировка качества от 10% до 100%
- Мгновенная конвертация без потери данных

### ⚙️ Обработка изображений
- 📏 Изменение размера (с сохранением пропорций)
- 🔄 Поворот на 90° влево/вправо
- 🪞 Отражение по горизонтали и вертикали
- 🔗 **Цепочка операций** — можно применять последовательно

### 🎨 Фильтры
- ⚫ Чёрно-белый (Grayscale)
- 🟤 Сепия (Sepia)
- 🔲 Инверсия цветов
- ☀️ Яркость (-100...+100)
- 🌓 Контраст (0...200%)
- 🌫️ Размытие (0...10px)
- ⟳ Сброс к оригиналу в один клик

### 💾 Скачивание
- Выбор формата при скачивании (PNG/JPEG/WEBP)
- Правильные расширения файлов (.jpg, а не .jpeg)
- Применение выбранного качества

## 🎯 Особенности

- ✅ **Полностью офлайн** — работает без интернета
- ✅ **Один HTML-файл** — никаких зависимостей и библиотек
- ✅ **3D-интерфейс** — красивые кнопки с эффектом нажатия
- ✅ **Drag & Drop** — перетаскивание файлов в окно
- ✅ **Адаптивный дизайн** — работает на телефонах и планшетах
- ✅ **Индикатор загрузки** — видно процесс обработки
- ✅ **Поддержка больших файлов** — до 10 МБ и более
- ✅ **Без рекламы и трекеров** — приватность на 100%

## 🚀 Быстрый старт

### Вариант 1: Локально
1. Скачайте файл `index.html`
2. Откройте его в любом браузере (Chrome, Firefox, Edge, Safari)
3. Готово! Работает без установки

### Вариант 2: GitHub Pages
1. Форкните или клонируйте репозиторий
2. Перейдите в **Settings → Pages**
3. Выберите ветку `main` и папку `/ (root)`
4. Через 1-2 минуты приложение будет доступно по адресу:
   ```
   https://ваш-логин.github.io/photo-converter-pro/
   ```

## 🛠️ Технологии

- **HTML5** — структура
- **CSS3** — 3D-эффекты, градиенты, backdrop-filter
- **Vanilla JavaScript** — без фреймворков
- **Canvas API** — обработка изображений
- **File API** — работа с файлами
- **Drag & Drop API** — перетаскивание
